# Komyut Backend Server

This folder contains the simple Node.js server for handling email sending.

## Setup Instructions

To run this server locally, follow these steps after cloning the main repository.

### 1. Navigate into this Directory

```bash
cd backend
```

### 2. Install Dependencies

```bash
npm install
```

### 2. Create Environment File (.env)

Create .env file in backend folder, copy this, and encode the credentials on Discord server.

GMAIL_USER=
GMAIL_APP_PASSWORD=

### 2. Run the Server

```bash
node index.js
```
